Argument and parameter:

def greet_with_name(name):  #name is the parameter
    print(f"HEllo {name}")

greet_with_name("Aditi") #"Aditi" is the argument

Positional argument and key word argument:
def greet_with(name, location):
    print(f"Hello {name}")
    print(f"What is it like in {location}")

greet_with("Aditi", "Milwaukee") #this is positional arguments,
# depends on which is given first will be the first parameter and the second argument is the second parameter

#This is keyword arguments, in which we mention the parameter , the argument associates with
greet_with(location="Wisconsin", name="Bob")
